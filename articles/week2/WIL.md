# WIL

2주차에는 학습 까지 진행하고 그래프로 성능 평가까지 확인하는 과정을 진행했다.
이제 좀 전체적인 흐름이 잡히기 시작했다. EPOCH나 그래프 같은 경우는 바꾸면 결과가 직관적으로 보이기 때문에 괜히 뿌듯했다. VOCAB이나 BATCH SIZE는 아직 눈에 띄게 차이를 못느껴서 기회가 된다면 값을 바꿔서 해봐야겠다. 하다가 코드가 deprecated된게 있어서 급하게 수정해서 넣었다. 최신화가 빠르게 이루어지나보다. 또 EPOCH를 올리니까 학습 시간이 꽤 길어져서, 모델이 조금 복잡해지고, EPOCH 수가 증가한다면 노트북이 못버티지 않을까 하는 걱정이 있다. 실제로 transformer로 학습하는데 2시간이 걸렸다.
